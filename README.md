This repository contains essential Java core concepts with clear explanations and practical examples. It is designed for interview preparation and strong fundamentals.

* Inheritance
* Polymorphism

* List (ArrayList, LinkedList)

* Map (HashMap)

### 3. Exception Handling

* try-catch
* finally
* throw & throws
* custom exceptions

### 4. Strings

* String vs StringBuilder vs StringBuffer
* Important methods
* Immutability

### 5. Multithreading (Optional)

* Thread creation
* Runnable interface
* Synchronization basics
