# String Methods in Java

The String class provides various methods to manipulate and process strings.

---

## 🔑 Common Methods

### 1. length()

Returns number of characters

---

### 2. trim()

Removes leading and trailing spaces

---

### 3. toUpperCase() / toLowerCase()

Changes case of characters

---

### 4. charAt()

Returns character at given index

---

### 5. substring()

Extracts part of string

---

### 6. contains()

Checks if substring exists

---

### 7. equals()

Compares string content

---

## 💡 Example

```java
String str = "Java";
str.length();
str.toUpperCase();
```

---

## 🔥 Interview Questions

### 1. Which method compares strings?

equals()

---

### 2. Difference between substring() and charAt()?

substring → returns string
charAt → returns character

---

## 🚀 Summary

String methods help in processing and manipulating text efficiently in Java.
