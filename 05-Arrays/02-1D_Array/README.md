# One-Dimensional Arrays in Java

A one-dimensional array is a linear collection of elements stored in a single row.

---

## 📌 Key Points

* Stores elements in a single dimension
* Accessed using a single index
* Index starts from 0

---

## 🔑 Traversal Methods

### 1. Using for loop

```java
for (int i = 0; i < arr.length; i++)
```

---

### 2. Using for-each loop

```java
for (int num : arr)
```

---

## 💡 Example

```java
int[] arr = {5, 10, 15, 20};
```

---

## 🔥 Interview Questions

### 1. What is 1D array?

Array with single dimension.

---

### 2. Difference between for and for-each?

| for loop   | for-each |
| ---------- | -------- |
| Uses index | No index |
| Flexible   | Simple   |

---

## 🚀 Summary

1D arrays are the simplest form of arrays used to store and access elements in a linear structure.
