# TreeMap Sorting using Comparable

Comparable is used to define natural sorting of keys inside the class.

---

## 🔑 Key Points

* Present in `java.lang`
* Used for default sorting
* Class must implement Comparable
* Uses `compareTo()`

---

## ⚙️ How it Works in TreeMap

* TreeMap sorts keys automatically
* Sorting is based on `compareTo()`

---

## 🔥 Interview Questions

### 1. What is Comparable?

An interface used for natural sorting.

---

### 2. Which method is used?

`compareTo()`

---

### 3. Where is it used?

* TreeMap
* TreeSet
* Collections.sort()

---

### 4. What does compareTo() return?

* 0 → equal
* Positive → greater
* Negative → smaller

---

## 🚀 Summary

Comparable is used when only one default sorting logic is required.
